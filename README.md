# Selangor Rainfall Application

This project is a web application for tracking and visualizing rainfall Json data in Selangor.

## Getting Started

Follow these steps to set up and run the project locally.

### 1. Install Node.js
[Download Node.js](https://nodejs.org/)

### 2. Git Clone
```bash
git clone https://github.com/danishfik/selangor-rainfall.git
```

### 3. Git Fetch
```bash
cd selangor-rainfall
git fetch
```

### 4. Git Pull
```bash
git pull origin main
```

### 5. Install Node Package
```bash
npm i
```

### 6. Generate Adonis App Key
```bash
node ace generate:key
```

### 7. Run Development Server (Local)
```bash
npm run dev
```

### 8. Access Local Host
[http://localhost:3333](http://localhost:3333)
